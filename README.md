# MotoGP Data Base Normalization

Este proyecto consiste en la modernización de la base de datos de **MotoGP Sports Entertainment Group**. El objetivo principal es transformar un sistema de almacenamiento desnormalizado y heredado en una base de datos relacional robusta bajo la tercera forma normal (3FN).

## Fases del Proyecto

1. **Diseño Conceptual:** Creación del modelo Entidad-Relación (E/R).
2. **Diseño Lógico:** Transformación al modelo relacional y normalización.
3. **Implementación:** Definición del esquema en MySQL.
4. **ETL / Migración:** Limpieza y carga de datos mediante Python (Pandas).
5. **Explotación:** Consultas SQL avanzadas para obtener estadísticas y campeones.

## Tecnologías utilizadas

* **Base de Datos:** MySQL 8.0
* **Lenguajes:** SQL, Python 3.x
* **Librerías:** Pandas, SQLAlchemy (para la migración).
* **Modelado:** MySQL Workbench
